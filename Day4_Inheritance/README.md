# Day4_Inheritance

This folder contains Python practice code for understanding **Inheritance** in Object-Oriented Programming (OOP).

## Contents

- [Inheritance.py](Inheritance.py): Core inheritance concepts and examples
- [ProblemsToSolve/Problem1.py](ProblemsToSolve/Problem1.py): Employee hierarchy (Employee, Developer, Manager)
- [ProblemsToSolve/Problem2.py](ProblemsToSolve/Problem2.py): Vehicle hierarchy (Vehicle, Car, Bike)
- [ProblemsToSolve/Problem3.py](ProblemsToSolve/Problem3.py): School hierarchy (Person, Student, Teacher)
- [ProblemsToSolve/Problem4.py](ProblemsToSolve/Problem4.py): Multilevel inheritance (Account, SavingsAccount, PremiumSavingAccount)
- [ProblemsToSolve/Problem5.py](ProblemsToSolve/Problem5.py): Multiple inheritance (Musician, Athlete, Person)

## Folder Structure

```text
Day4_Inheritance/
|-- Inheritance.py
`-- ProblemsToSolve/
    |-- Problem1.py
```

## Requirements

- Python 3.8 or above
- No external packages required

## Run the Files

Open terminal in this folder and run:

```bash
python Inheritance.py
python ProblemsToSolve/Problem1.py
python ProblemsToSolve/Problem2.py
python ProblemsToSolve/Problem3.py
python ProblemsToSolve/Problem4.py
python ProblemsToSolve/Problem5.py
```

For PowerShell (Windows):

```powershell
python .\Inheritance.py
python .\ProblemsToSolve\Problem5.py
```

## Learning Goals

After completing this folder, you should be able to:
- Create parent and child classes
- Use method overriding
- Use `super()` in constructors and methods
- Understand single, multiple, and multilevel inheritance
- Understand Method Resolution Order (MRO)
